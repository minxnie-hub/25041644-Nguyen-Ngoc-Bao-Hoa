# 25041644-Nguyễn-Ngọc-Bảo-Hoà

Portfolio học phần **Công nghệ số và AI** của **Nguyễn Ngọc Bảo Hoà**.

## Cấu trúc
- `index.html`: Trang chủ
- `contact.html`: Trang liên hệ và thông tin học tập
- `projects/`: 6 trang chi tiết cho từng bài tập
- `assets/docs/`: Tệp PDF gốc
- `assets/img/`: Ảnh minh chứng và ảnh thumbnail

## Thiết kế
Website được phát triển theo concept **“Hành trình qua ngôn ngữ và khung hình”**, lấy cảm hứng từ sở thích đi du lịch và xem phim của sinh viên, kết hợp tinh thần học thuật, ngôn ngữ và công nghệ.

## Deploy GitHub Pages
1. Tạo repository mới trên GitHub.
2. Upload toàn bộ source lên nhánh `main`.
3. Vào **Settings → Pages** và chọn nguồn **GitHub Actions**.
4. Workflow trong `.github/workflows/static.yml` sẽ tự động deploy.
